Ferramenta de Conteúdos Delta

Guia prático para criar e automatizar conteúdos profissionais usando N8N e Canva na Delta Academy.

Criado por glaucokelly
01
O que é o N8N

O N8N é uma plataforma de automação de workflows open-source que permite conectar diferentes aplicativos e serviços sem precisar programar. Pense nele como um "gerente invisível" que executa tarefas repetitivas por você — 24 horas por dia, 7 dias por semana.

Na Delta Academy, o N8N é usado para automatizar toda a cadeia de produção e distribuição de conteúdo, desde a captação de ideias até a publicação nas redes sociais.

Principais usos na Delta
  • Automatizar a criação de artes e posts
  • Integrar formulários com planilhas e ferramentas de design
  • Agendar e distribuir conteúdo em múltiplas plataformas
  • Monitorar métricas e gerar relatórios automáticos
  • Conectar leads com fluxos de e-mail e WhatsApp
02
O que é o Canva

O Canva é uma ferramenta de design gráfico online que permite criar artes profissionais para redes sociais, apresentações, vídeos, e-books e muito mais — sem precisar de experiência em design.

Com templates prontos e uma interface intuitiva de arrastar e soltar, qualquer membro da Delta Academy consegue produzir conteúdo visual de alta qualidade em poucos minutos.

Principais usos na Delta
  • Criação de posts para Instagram, LinkedIn e YouTube
  • Design de apresentações e materiais de aula
  • Thumbnails para vídeos e cursos
  • E-books, guias e materiais ricos para leads
  • Templates padronizados com a identidade visual Delta
03
Como usar o N8N — Passo a Passo

Siga os passos abaixo para criar seu primeiro workflow de automação de conteúdo no N8N:

  1. Acesse a plataforma N8N — entre no painel da Delta Academy e abra o N8N
  2. Crie um novo workflow — clique em "New Workflow" e dê um nome descritivo
  3. Adicione um trigger (gatilho) — escolha o que vai iniciar o fluxo: um formulário preenchido, horário agendado, webhook ou planilha atualizada
  4. Conecte os nodes — adicione as ações que o workflow vai executar (buscar dados de planilha, chamar API do Canva, enviar mensagem)
  5. Teste o fluxo — clique em "Execute Workflow" para ver o resultado de cada etapa
  6. Ative o workflow — quando estiver funcionando, ative o toggle para ele rodar automaticamente
Exemplo de estrutura de workflow de conteúdo
Trigger: Planilha Google atualizada com novo tópico
→ Node 1: Buscar dados do tópico (título, descrição, formato)
→ Node 2: Chamar API do Canva para gerar arte
→ Node 3: Salvar arte na pasta do Google Drive
→ Node 4: Agendar post na plataforma de mídia social
→ Node 5: Notificar time no WhatsApp com link da arte
04
Como usar o Canva — Passo a Passo

Veja como criar artes profissionais rapidamente no Canva usando os templates e padrões visuais da Delta Academy:

  1. Acesse o Canva — entre em canva.com com o e-mail da Delta Academy
  2. Abra a pasta da Delta — todos os templates oficiais estão organizados na pasta compartilhada do time
  3. Escolha o template — selecione o formato: post quadrado, story, carrossel, apresentação ou thumbnail
  4. Edite o conteúdo — substitua os textos e imagens mantendo as cores e fontes oficiais da marca
  5. Revise antes de exportar — verifique ortografia, alinhamento e se todos os elementos estão na área segura
  6. Exporte no formato certo — PNG para imagens estáticas, MP4 para animações, PDF para materiais impressos
Checklist rápido antes de publicar
[ ] Texto revisado sem erros ortográficos
[ ] Cores dentro do padrão Delta Academy
[ ] Logo da Delta visível e no lugar correto
[ ] Resolução adequada (1080x1080 para feed, 1080x1920 para story)
[ ] CTA (chamada para ação) claro e presente
[ ] Arquivo exportado no formato correto
05
Integrando N8N + Canva: O Pipeline de Conteúdo Delta

A maior vantagem de usar N8N e Canva juntos é criar um pipeline de conteúdo automatizado: o N8N alimenta o Canva com dados e o Canva entrega as artes prontas para publicação — tudo sem intervenção manual.

Usando a API do Canva, o N8N consegue:

  • Criar novas artes a partir de templates existentes
  • Preencher campos de texto automaticamente com dados de planilhas
  • Exportar as artes em diferentes formatos e resoluções
  • Organizar os arquivos gerados em pastas específicas
Fluxo completo de conteúdo automatizado
1. Responsável preenche planilha com: tópico, data, formato e público-alvo
2. N8N detecta nova linha na planilha (trigger automático)
3. N8N chama API do Canva: seleciona template e insere título e descrição
4. Canva exporta a arte gerada automaticamente
5. N8N salva no Google Drive na pasta correta
6. N8N agenda o post na ferramenta de mídia social
7. Time recebe notificação via WhatsApp com preview e link de aprovação
06
Dicas e Boas Práticas
N8N
  • Sempre nomeie seus workflows de forma descritiva (ex: "Geração Arte Instagram — Novo Tópico")
  • Use o recurso de "error workflow" para receber notificações quando algo falhar
  • Documente cada workflow com uma nota explicando o que ele faz
  • Nunca delete um workflow ativo — desative-o primeiro e aguarde 24h
  • Use variáveis de ambiente para armazenar tokens e chaves de API com segurança

Canva
  • Salve os templates oficiais como "favoritos" para acessar rapidamente
  • Use a paleta de cores salva da Delta para garantir consistência visual
  • Ative o recurso "Brand Kit" para manter fontes, cores e logos sempre à mão
  • Prefira exportar em PNG com fundo transparente quando usar sobre outras imagens
  • Mantenha a pasta do time organizada por mês e formato de conteúdo
